# Task 1: Iris Dataset Exploration and Visualization

## 📌 What I Did
I explored the famous Iris dataset to understand patterns in flower measurements.

## 📊 Dataset
- **Source**: Iris dataset (built into seaborn)
- **Samples**: 150 iris flowers
- **Features**: 4 (sepal length, sepal width, petal length, petal width)
- **Target**: 3 species (Setosa, Versicolor, Virginica)

## 🛠️ Tools Used
- Python
- pandas
- matplotlib
- seaborn

## 📈 Visualizations Created
- Scatter plots
- Histograms
- Box plots
- Correlation heatmap
- Pair plot

## 🔍 Key Findings
- Setosa is easily distinguishable (has smallest petals)
- Petal features are best for separating species
- No missing values in dataset
- Strong correlation between petal length and width
